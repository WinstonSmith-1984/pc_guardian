3. Instructions 

    How to run the LOTL-LAN Sandbox:

        System Prerequisite: Ensure you have Wireshark (TShark) installed on your computer, as this app needs it to capture live network traffic.

        Run the Script: Copy the setup script into your project folder and run it.

        Why use the Sandbox? This creates an isolated "environment" for the app. It prevents the network analysis libraries from cluttering your computer's main settings and ensures the app has the exact versions of pyshark and plotly it needs to visualize your traffic.
